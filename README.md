The **Add to Cart** button is available on every product page via Wix Store. Customers can view their cart at any time using the cart icon in the navigation header.

## ⚠️ Challenges
 
1. **Choosing the right template:** There are hundreds of Wix templates available. Selecting one that matched the beauty aesthetic while being easy to customize required time and comparison.
2. **Product catalog setup:** Configuring the Wix Store module — adding products, setting prices, writing descriptions, and uploading images — was the most time-consuming part of the project.
3. **Design consistency across pages:** Maintaining a consistent color palette, typography, and visual style across all pages (Homepage, Shop, About, Contact) was challenging but essential for a professional result.
4. **Image sourcing:** Finding free, high-quality beauty product images that fit the store's identity required searching through multiple sources such as Unsplash and the Wix Media Library.
---
 
## 💡 Lessons Learned
 
- **No-code platforms are powerful:** Wix allows anyone to build a fully functional, visually appealing e-commerce website without writing code — significantly lowering the barrier to entry for entrepreneurship.
- **UX design matters:** Clear navigation, consistent branding, and well-placed call-to-action buttons directly impact how users interact with a website.
- **Content is key:** Well-written product descriptions and a compelling brand story make a store feel more professional and trustworthy.
- **GitHub & Markdown are essential skills:** Documenting a project with structured Markdown in a public repository is a core professional habit in tech, design, and development fields.
- **Planning saves time:** Preparing the content (product names, descriptions, images) before opening the editor made the building process much faster and more focused.
